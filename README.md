# GDELT-Automotive-Media-Analysis

Data engineering pipeline for collecting and processing automotive industry news from the GDELT Web News database. Built as part of a Social Networks Analysis project examining Chinese EV brands vs established automotive leaders in trade media discourse.

**Author:** Jakub Wojciechowski (data pipeline contribution)  
**Team:** Group 3 ‚Äî Kozminski University  
**Course:** Social Networks Analysis  
**My contribution:** data collection pipeline (Steps 1‚Äì3 below)

---

## 1. Research context

**Research question:** Can Chinese EV brands (BYD, NIO, XPeng) compete with established automotive leaders (BMW, Ford, Tesla) in automotive trade media discourse?

**Method:** Semantic Brand Score (SBS) ‚Äî measuring brand Prevalence, Diversity, and Connectivity in the semantic network of collected articles.

**Key finding:** Legacy brands (Ford, BMW) still dominate the semantic space. BYD is entering the discourse but does not yet control it.

## 2. Data pipeline (my contribution)

The pipeline consists of three steps, implemented in separate notebooks in the `Code/` folder:

### Step 1 ‚Äî Download (`Code/01_download.ipynb`)
Downloads GDELT Web NGrams files for the target time range using the `gdeltnews` library.

- **Time range:** 2026-02-03 to 2026-02-05
- **Files downloaded:** 534 `.gz` files ‚Üí `gdeltdata/`

### Step 2 ‚Äî Reconstruct (`Code/02_reconstruct.ipynb`)
Decompresses and reconstructs full article text. Filtered to English-language content from 5 automotive trade portals.

- **Portals:** motortrend.com, caranddriver.com, autocar.co.uk, autonews.com, motor1.com
- **Output:** `gdeltpreprocessed/`

### Step 3 ‚Äî Filter & merge (`Code/03_filter_merge.ipynb`)
Applies brand keyword filter and deduplicates by URL.

- **Brands:** BYD, NIO, XPeng, Geely, SAIC, MG, Tesla, BMW, Audi, Mercedes, Ford
- **Output:** `final_data_kuba.csv` ‚Äî 63 deduplicated articles
